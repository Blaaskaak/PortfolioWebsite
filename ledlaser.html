    Sound reactive system

This installation combines 24 laser diodes with six strips of 70 LEDs to create an immersive lighting setup. The whole system is sound-reactive, translating audio input into dynamic patterns and intensity changes for an immersive, responsive light experience.

LIGHT INSTALLATION
PROJECT DETAILS

For this project, I designed and built an interactive light installation featuring 24 lasers and six LED tubes, each containing 70 LEDs. Each laser is individually connected to its own MOSFET and controlled by a single microcontroller, allowing precise, independent control. To diffuse the LED light, I repurposed old fluorescent tubes, giving the piece a sustainable element. Both the LEDs and lasers are integrated with TouchDesigner, enabling real-time, sound-reactive visuals driven by live beat analysis.
